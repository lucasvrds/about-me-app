# 📱 CodeCenter

Este é um aplicativo mobile desenvolvido com **Expo** e **React Native** para o Trabalho Prático. O projeto foca na estruturação de componentes, estilização de interfaces e navegação entre telas.

## 👥 Integrantes do Grupo
* Lucas Vieira
* Vitor Soler

## 📲 Descrição do App
O CodeCenter é composto por duas telas principais:

1.  **🏠 Home:** Apresenta a logo do app, imagem de perfil e um texto de apresentação. Inclui botões para navegação ("Sobre mim"), um alerta de contato ("Entre em contato!") e uma seção de redes sociais (GitHub, LinkedIn, Instagram e Twitch). Também possui um controle de alternância (*switch*) que exibe um alerta ao ser acionado.
2.  **👤 Sobre mim:** Contém um título, um texto descritivo detalhando a biografia/apresentação e um rodapé com os créditos do projeto.

## 🖥️ Tecnologias

## Como rodar o projeto

1. **Clone o repositório:**
    
    https://github.com/lucasvrds/about-me-app.git
    
2. **Instale as dependências:**
    
    `npm install`
    `npx expo install react-native-screens react-native-safe-area-context`

3. **Inicie o Expo**
    
    `npx expo start`
